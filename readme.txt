Welcome to the NightGazr README FILE

It contains some info that you might need when working with this theme.

The official project site is at https://code.google.com/p/nightgazr/.

The documentation is at https://code.google.com/p/nightgazr/w/list.

The live website/preview is at http://nightgazr.tumblr.com/ (also the actual dev site - might break from time to time).

Support is not yet available for 1.0 alpha.

Bug reports; and feature and enhancement requests/proposals go to https://code.google.com/p/nightgazr/issues/list.

Open for forking. Just do a checkout and commit it to your own repo.

Open for contributions. Project is maintained with Git.

Get the latest build at https://code.google.com/p/nightgazr/source/list.
